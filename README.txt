Steps to run:

1. Install latest LTS version of Node.js
    - https://nodejs.org/en/download

2. Copy/Paste the full `HarvestApiClient` folder to your local Documents directory in a new Projects folder
    - `"C:\Users\<your-user-name>\Documents\Projects\"`

3. Change the environment variables to match your Harvest API credentials
    - You'll need a new personal access token + account ID if you haven't made one yet
        - https://id.getharvest.com/developers

4. If needed, customize the time entry values within `updateWeekTimesheet.js`

5. Run the `InitializeTimesheet.bat` script
